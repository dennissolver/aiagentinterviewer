# AI Interview Platform

AI-powered voice interview platform built with Next.js, Supabase, and ElevenLabs.

## Configuration

This platform was provisioned automatically. Your environment variables are configured in Vercel.

## Environment Variables

```
NEXT_PUBLIC_SUPABASE_URL=your-supabase-url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your-anon-key
SUPABASE_SERVICE_ROLE_KEY=your-service-role-key
ELEVENLABS_API_KEY=your-elevenlabs-api-key
NEXT_PUBLIC_PLATFORM_NAME=Your Platform Name
NEXT_PUBLIC_COMPANY_NAME=Your Company Name
```

## Getting Started

1. Clone this repository
2. Install dependencies: `npm install`
3. Set up environment variables
4. Run development server: `npm run dev`
